# 20. Approved Tools Policy

Aluna utilizes a suite of approved software tools for internal use by workforce members. Use of other tools requires approval from Aluna leadership.

## 20.1 List of Approved Tools

* **GitHub**. GitHub is hosted version control platform. It is utilized for storage of configuration scripts and other infrastructure automation tools, as well as for source and version control of application code used by Aluna.
* **Box**. Box is used for storage of files and sharing of files with Partners and Customers.
* **Google Apps**. Google Apps is used for email and document collaboration.
